Base de Datos

MySql


  1. Crear una base de datos

    2. CREATE DATABASE IF NOT EXISTS [db_name] CHARACTER SET = [charset] DEFAULT COLLATE = [collate]
    Ejemplo
    CREATE DATABASE IF NOT EXISTS base_prueba CHARACTER SET = utf8 DEFAULT COLLATE = utf8_general_ci

  2. Como hacer respaldos de una base de datos

    3. 1) mysqldump [opciones] nombre_bd [nombre_tabla1 nombre_tabla2 ...] > respaldo.sql
    2) mysqldump [opciones] --databases nombre_bd1 nombre_bd2 > respaldo.sql
    3) mysqldump [opciones] --all-databases > respaldo.sql
    Ejemplo
    mysqldump --lock-tables --single-transaction --default-character-set=utf8 -u usuario -p contraseña -h localhost amazon_DB > ExportFact04122017.sql >

  3. Como crear Usuario en base de datos

    4. CREATE USER 'Nuevo_usuario'@'localhost' IDENTIFIED BY 'Contraseña_de_BD';>

  4. Quitar a un usuario

    5. DROP USER ‘usuario_prueba’@‘localhost’;>

  5. Otorgar privilegios a un usuario
    6. -Todos los privilegios

    ALL PRIVILEGES: como mencionamos previamente esto permite a un usuario de MySQL acceder a todas las bases de datos asignadas en el sistema.
    CREATE: permite crear nuevas tablas o bases de datos.
    DROP: permite eliminar tablas o bases de datos.
    DELETE: permite eliminar registros de tablas.
    INSERT: permite insertar registros en tablas.
    SELECT: permite leer registros en las tablas.
    UPDATE: permite actualizar registros seleccionados en tablas.
    GRANT OPTION: permite remover privilegios de usuarios.

    GRANT [permiso] ON [nombre de bases de datos].[nombre de tabla] TO ‘[nombre de usuario]’@'localhost’;
    grant all privileges ON `Nombre_de_BD`.* TO 'Usuario_BD'@'%' IDENTIFIED BY 'Contraseña_BD';

    Siempre Actualizar los privilegios con un flush privileges


  6. Quitar privilegios a un usuario

    7. REVOKE [permiso] ON [nombre de base de datos].[nombre de tabla] FROM ‘[nombre de usuario]’@‘localhost’;>
